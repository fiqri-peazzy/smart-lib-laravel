1. Cara Menjalankan Project

Ikuti langkah-langkah di bawah ini untuk menjalankan sistem di komputer lokal Anda. Panduan ini dirancang agar mudah diikuti bahkan jika Anda bukan seorang programmer.

1.1 Persiapan Software

  • XAMPP atau Laragon: Untuk menjalankan server database.
  • Visual Studio Code: Untuk membuka folder project.
  • PHP 8.2+: Versi PHP minimal yang dibutuhkan.

1.2 Langkah-langkah Detail

Langkah 1: Jalankan Server (Laragon/XAMPP)

Buka Laragon atau XAMPP Anda. Klik tombol Start All atau pastikan layanan Apache dan MySQL sudah menyala (berwarna hijau).

Langkah 2: Buka Folder di VS Code

Buka Visual Studio Code, klik menu File > Open Folder, lalu pilih folder smart-lib.

Langkah 3: Buka Terminal Bash

Di VS Code, tekan tombol Ctrl + ` (backtick) untuk membuka terminal. Pastikan di pojok kanan bawah terminal tertulis bash. Jika tidak, klik tanda panah ke bawah dan pilih Git Bash.

Langkah 4: Jalankan Perintah Setup

Ketik perintah berikut satu per satu dan tekan Enter:

composer install
php artisan key:generate
php artisan migrate --seed
npm install
npm run dev

Langkah 5: Akses Aplikasi

Buka terminal satu lagi, jalankan perintah:

php artisan serve

Lalu buka browser (Chrome/Edge) dan ketik: http://localhost:8000

3. Alur Sistem untuk Pengguna Awam

Sistem ini dirancang untuk memudahkan proses pinjam-meminjam buku di perpustakaan. Berikut adalah gambaran alurnya secara sederhana.

3.1 Alur Peminjaman Buku

Cari Buku di Katalog ➔ Klik Request ➔ Datang ke Perpustakaan ➔ Scan QR Code ➔ Selesai
  1. Cari: Anda mencari buku yang diinginkan di website.
  2. Pesan: Setelah ketemu, klik tombol Pinjam. Anda akan mendapat tiket berupa QR Code.
  3. Ambil: Datang ke perpustakaan, tunjukkan tiket tadi ke petugas.
  4. Verifikasi: Petugas menscan tiket Anda, dan memberikan bukunya.

3.2 Alur Pengembalian Buku

Bawa Buku ke Petugas ➔ Petugas Scan Buku ➔ Selesai (Bayar Denda jika Telat)

Sangat mudah! Cukup bawa bukunya, petugas akan menscan barcode di buku, dan status pinjaman Anda akan otomatis selesai.

⚠️ Info Penting tentang Denda:

Jika Anda telat mengembalikan, sistem akan memberikan denda Rp 5.000 per hari. Anda tidak bisa meminjam buku lagi sebelum denda dilunasi.

5. Modul Katalog & Koleksi Digital

Halaman utama adalah tempat Anda mencari semua koleksi, baik buku fisik maupun buku digital (E-book).

5.1 Mencari Buku Fisik

Cukup ketik judul buku di kolom pencarian. Anda bisa melihat status buku apakah Available (tersedia) atau sedang dipinjam orang lain.

[Gambar: Katalog Utama Smart-Lib]

5.2 Akses Koleksi Digital (E-Book)

Selain buku fisik, Anda bisa membaca buku digital secara langsung di website.

  • Masuk ke menu Digital Library.
  • Pilih buku yang ingin dibaca.
  • Klik tombol Read Online untuk membuka viewer PDF bawaan.
  • Anda juga bisa mendownload buku tersebut jika diizinkan oleh admin.

8. Modul Admin Panel (Filament)

Modul ini digunakan oleh petugas perpustakaan (Staff) dan Admin untuk mengelola data data perpustakaan.

8.1 Cara Menambah Buku (Oleh Admin)

  1. Buka alamat http://localhost:8000/admin dan login.
  2. Pilih menu Books di sebelah kiri.
  3. Klik tombol New Book di pojok kanan atas.
  4. Isi Nama Buku, ISBN, dan upload Cover Image.
  5. Klik Create untuk menyimpan.

💡 Tip untuk Admin:

Anda bisa melihat statistik peminjaman paling populer di Dashboard Admin untuk membantu memutuskan pengadaan buku baru.
